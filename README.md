### Get started

1. Clone the repo
2. Set the `.env` variables present in `.env.sample`
```bash
cp .env.sample .env
```
3. Build & start the container
```bash
docker compose up --build
```
4. Access at [localhost (port 5000 binded to 80)](http://localhost/login)
